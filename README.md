# coarray-fortran-examples

Coarray-fortran examples for beginners by a beginner.

## How to Run

### exe

```bash
mkdir build
cd build
cmake ..
make
```

### unit test

```bash
cmake -B build
cmake --build build
cmake --build build --target test
```
